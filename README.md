:fork_and_knife:Spoon
======================

Plate의 Web Scraper입니다.

설치
-----

```
$ git clone https://github.com/joshua1b/Spoon.git
```

실행
-----

```
$ python -m venv myvenv
$ source myvenv/bin/activate
$ pip install -r requirements.txt
$ python main.py [year] [month]

```
